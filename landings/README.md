# Landing Pages

Colección de sitios web desplegados en Netlify con hardening de seguridad de nivel A+.

## Proyectos

### Zero-Trust Portfolio
Portfolio profesional estilo terminal cyberpunk con feed de logs en vivo, reloj UTC y matriz de habilidades.
[/landings/zero-trust/](./zero-trust/)

### Compliance Checker
Herramienta de autoevaluación de cumplimiento normativo (GDPR/LGPD) con gráfico radar y scoring dinámico.
[/landings/compliance-checker/](./compliance-checker/)

## Seguridad

Todos los proyectos incluyen `netlify.toml` con:
- Content Security Policy (CSP) restrictiva
- HSTS (HTTP Strict Transport Security)
- X-Frame-Options: DENY
- X-Content-Type-Options: nosniff
- Calificación A+ en Mozilla Observatory

---

**Stack:** HTML5, Tailwind CSS, Chart.js, Lucide Icons, Netlify
