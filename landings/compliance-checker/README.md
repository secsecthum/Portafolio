⚖️ Compliance Center | Global Security Auditor 🛡️Este proyecto es una herramienta de Auditoría de Cumplimiento Normativo diseñada para evaluar la postura de seguridad de una infraestructura frente a estándares internacionales. Utiliza una arquitectura basada en el cliente para garantizar la privacidad total de los datos declarados.🧠 ¿Qué hace esta herramienta?El Compliance Center permite a los oficiales de seguridad (CISO) y desarrolladores realizar un escaneo manual de sus controles preventivos y detectivos. El motor de análisis procesa la entrada y genera:📊 Puntuación de Postura Dinámica: Un porcentaje real basado en pesos críticos de seguridad.📈 Gráfico de Radar de Riesgos: Desglose visual de las áreas de Identidad, Datos y Red.🔍 Reporte Técnico de Hallazgos: Recomendaciones automáticas basadas en las vulnerabilidades detectadas.🛠️ Stack TecnológicoFrontend: HTML5, Tailwind CSS (Cyberpunk UI Design).Gráficos: Chart.js para visualización de datos complejos.Iconografía: Lucide-Icons para una interfaz limpia y técnica.Seguridad: netlify.toml con configuración avanzada de Content Security Policy (CSP).🚀 Ejecución en Local (Quick Start)Para visualizar y probar el auditor en tu entorno local, puedes utilizar los siguientes métodos:Opción 1: Usando Python (Recomendado)Ideal para sistemas Linux/macOS o Windows con Python instalado:Bash# Entra al directorio del proyecto
cd compliance-checker/components

# Inicia el servidor
python3 -m http.server 8000
Luego abre tu navegador en: http://localhost:8000Opción 2: Extensión de VS CodeSi usas Visual Studio Code, puedes usar la extensión Live Server:Haz clic derecho sobre index.html.Selecciona "Open with Live Server".📑 Metodología de EvaluaciónEl algoritmo evalúa 6 pilares fundamentales de la ciberseguridad moderna:CategoríaControlImpacto en ScoreIdentidadMulti-Factor Auth (MFA)Crítico (60%)IdentidadIAM Roles (RBAC)Alto (40%)DatosCifrado AES-256Crítico (70%)DatosBackups InmutablesMedio (30%)RedWAF (Web Application Firewall)Alto (60%)RedCentralización de Logs (SIEM)Bajo (40%)🛡️ Seguridad y Cumplimiento del SitioEste proyecto no solo audita, sino que predica con el ejemplo. El despliegue incluye:X-Frame-Options: Protege contra ataques de Clickjacking.HSTS: Fuerza el uso de HTTPS.CSP (Content Security Policy): Evita la inyección de scripts maliciosos (XSS).Nota de Privacidad: Este auditor opera bajo el principio de Local Analysis Mode. Ningún dato ingresado en el formulario es enviado a servidores externos; todo el cálculo se realiza en la memoria volátil de tu navegador. 🔐
