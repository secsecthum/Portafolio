# Servicios Backend — APIs Full-Stack

Aplicaciones Node.js/Express con frontends integrados, diseñadas con estándares de seguridad y estética cyberpunk.

## Proyectos

### Gold Terminal — Crypto Market Intelligence
Dashboard de análisis de criptomonedas con Fear & Greed Index y predicciones de precios.
[/services/Bitcoins/](./Bitcoins/)

### LexTech Elite — Legal Intelligence Platform
Agregador de legislación multiplaís (LGPD, GDPR) con consultas a APIs gubernamentales.
[/services/LGPD/](./LGPD/)

### ShieldScan — Domain Security Auditor
Herramienta OSINT de auditoría de seguridad web con scoring dinámico de riesgo.
[/services/Security-Scan-Report/](./Security-Scan-Report/)

## Stack Común

- **Backend**: Node.js, Express 5
- **Seguridad**: Helmet, CORS, Rate Limiting
- **Frontend**: HTML5, Tailwind CSS, Chart.js, Lucide Icons
- **Cliente HTTP**: Axios
